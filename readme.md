## Descrição do sistema

Este projeto é um sistema de controle financeiro pessoal que permite ao usuário:

- Autenticar-se com conta Google;
- Gerenciar suas transações (CRUD: criar, listar, editar e excluir);
- Visualizar um dashboard com resumo (saldo, entradas, saídas);
- Manter sessão do usuário usando Pinia;
- Navegar entre rotas protegidas (login, dashboard, transações, perfil).

O objetivo da entrega é cumprir os requisitos da avaliação: Frontend em Vue 3 (Vuetify, Vue Router, Pinia), backend em Express.js com persistência (Supabase/Postgres ou MySQL), autenticação Google, proteção de rotas e integração via API REST.

## Tecnologias

- Frontend: Vue.js 3, Vite, Vuetify, Vue Router, Pinia
- Backend: Node.js, Express.js
- Persistência: Supabase (PostgreSQL) 
- Autenticação: Supabase Auth (Google) ou Firebase Authentication (Google)
